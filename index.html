HTML INTRODUCTION.

<DOCTYPE html> defines that doc is html doc. It is the document type declaration. It must appear only once. It is not case sensitive.
<html> is the root element of the html page. The HTML document begins with <html> and ends with </html>.
<title> element specifies name of browser's title bar or tab bar.
<body> elemnt contains all the visible thing of website.
ELEMENTS:
syntax: <tagname>content</tagname>
start tag content end tag
<h1> imp heading </h1>
<p> paragraph </p>
Some elements doesn't have an ending tag neither content. These are called as Empty elements. for example: <br>.
WEB BROWSERS read the HTML documents and display them correctly.

HTML BASIC.

HTML HEADINGS:
Headings are ranked from 1 to 6 based on its importance. <h1> is the most important one and can only be used once.
<h6> is the least impostant.
HTML PARAGRAPHS:

Paragraphs are defined with the <p> tag.
HTML LINKS:

Links are defined with the <a> tag.
For example:
This link will take you to w3 schools.
<a href="upload your link here"> The blue thing is written here. </a>.
The "href" attribute is used for providing link's destination. Attributes are used for providing extra information about elements.
HTML is not case sensitive. so <p> and <P> both mean the same!
